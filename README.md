### Run Locally
1. Clone the project repository:
```
git clone https://github.com/carlywilk/photographer-sample.git
```

2. Go to the project directory:
```
cd photographer-sample
```

3. Install dependencies:
```
npm install
```

4. Run the project:
```
npm start
```
